# Tierno News
API REST con [Spring Boot](https://spring.io/projects/spring-boot), Spring Data JPA y MySQL.
Aplicación sencilla con artículos y autorías.
Para el ciclo de Desarrollo de Aplicaciones Multiplataforma.
## Modelos
### Article
Campos:
- id
- title
- content
- date
- author
### Author
- id
- name
- age

## Endpoints
### Article
    GET /article
    GET /article/{id}
    POST /article
    PUT /artible/{id}
